Tamam Rides Car Rental

Description

Tamam Rides is an innovative car rental Android application developed as part of a university assignment collaboration among three individuals, including myself. Leveraging Dart and Flutter, we crafted this user-friendly car rental app that seamlessly integrates the Google Maps API to provide users with real-time map data for efficient pick-ups and drop-offs.
